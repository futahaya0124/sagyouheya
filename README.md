# 🔧 ガンプラ作業通話募集Bot

毎日18時に自動で作業通話の募集メッセージを投稿するDiscord Botだよ！

## 機能
- 毎日18時（日本時間）に自動投稿
- 4つのボタンで参加時間を選べる
  - 🌙 21時頃から
  - 🌃 22時頃から
  - 🌛 23時以降
  - 😴 今日はやらない
- ボタンを押すと参加者一覧がリアルタイム更新

## Railwayへのデプロイ手順

### 1. GitHubリポジトリを作成
1. https://github.com/new でリポジトリ作成
2. このフォルダの中身をアップロード

### 2. Railwayでデプロイ
1. https://railway.app にアクセス
2. GitHubアカウントでログイン
3. 「New Project」→「Deploy from GitHub repo」
4. 作成したリポジトリを選択
5. デプロイが始まる（最初は失敗するけど大丈夫！）

### 3. 環境変数を設定
1. Railwayのプロジェクト画面で「Variables」タブをクリック
2. 以下の2つを追加：
   - `DISCORD_TOKEN`: BotのトークンをDeveloper Portalからコピー
   - `CHANNEL_ID`: 投稿先チャンネルのID

### 4. チャンネルIDの取得方法
1. Discordの設定→詳細設定→「開発者モード」をON
2. 投稿したいチャンネルを右クリック
3. 「チャンネルIDをコピー」

### 5. 動作確認
環境変数を設定すると自動で再デプロイされるよ！
Railwayの「Deployments」タブでログを確認してね。

## 手動で投稿をテストしたい場合
Railwayにデプロイした後、一度Botを再起動すれば動くよ。
18時を待たずにテストしたい場合は、index.jsのcron設定を一時的に変更してね。
